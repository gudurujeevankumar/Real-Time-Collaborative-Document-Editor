# scribe-flow-collab

Lovable Generated Project

This is a real-time collaborative document editor built with React, Vite, Supabase, and Tailwind CSS.

## Features
- Real-time collaboration
- Rich text editing
- Document sharing
- Activity log
- User authentication

## Getting Started
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used
- React
- Vite
- Supabase
- Tailwind CSS
